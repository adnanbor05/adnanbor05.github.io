Wedding Invitation

This project is designing a website application for the wedding invitation of Khabib & Nanik.

The Problem

khabib & nanik is a friend of mine who asked me for help in making a wedding invitation on a website so that the spread of invitations would be easier and wider.

Fiture

I design the homepage design from the draft and develop it. Then i also design all the inner pages. The list of the pages that i designed was :
  1. Home
  2. Couple
  3. Event
  4. Countdown
  5. Galery
  6. Message

Home



Home page is the main page of the wedding invitation website.

Couple



Couple page is displays the profiles and backgrounds of the two couples who are getting married.

Event



Event page is displays the place and time of the wedding.

Countdown



Countdown is a page that displays the wedding date countdown

Galery



This is a page that displays a photo gallery of the couple's wedding.

Message



This is a page that displays page provided by invited guests to send greetings to the bride and groom.
